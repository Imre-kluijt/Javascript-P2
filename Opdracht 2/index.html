Opdracht 2

1: Wat is het verschil met de script tag boven in te laden in de tag t.o.v onder inladen aan het einde van je HTML tag?

Als je de script tag boven aan invoert dan wordt het script uitgevoerd voordat de HTML van de pagina is geladen.

2: Wat doet de attribuut “defer” in de script tag?

Het defer-attribuut in een script-tag zorgt ervoor dat je script pas wordt uitgevoerd nadat de hele HTML is geladen, maar zonder de pagina-opbouw te blokkeren.

3: Wat is het verschil tussen HTML en JavaScript?

HTML bepaalt wat er op de webpagina staat (de structuur en inhoud), terwijl JavaScript bepaalt wat de pagina doet (interactiviteit en gedrag). HTML is een opmaaktaal; JavaScript is een programmeertaal die de pagina dynamisch maakt.
